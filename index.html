Schema

    'type' : 'object',

        'name' : {
            'type' : 'string'
        }
    }
}

{
    name: "Willi"
}

Json Rest Api

  • Dokumentation von Url, Verb, Parameter und Rückgabetypen
  • Test-Client zum Testen der API ohne Anwendung

Swagger

  • Spezifikation einer Rest API
  • wie WSDL oder WADL
  • generischer Api-Client wie SoapUI
  • Viele Bibliotheken für Express, Spring MVC, Ruby u.a.

Administration

  • Administration für persistente Daten
  • Nutzer sind Berater, Kunden
  • Generischer Datenbankbrowser
  • Aufwand muss möglichst klein sein

Lösung

  • Generieren des UI anhand von existierenden Metadaten
  • Java-Klassen, Datenbank-Schema, Mongoose-Schema, ..

Validation

{
    'type' : 'object',
    'properties': {
        'name' : {
              'type':'string'
              'pattern':'[a-zA-Z]*'
             },
        'email' :  {
              'type':'string'
              'format':'email'
              },
    },
    'required' : ['name']
}

Recursive data structures


{
    'type' : 'object',
    'properties': {
        'name' : {'type': 'string'}
        'friend' : {'$ref': '#'}
    }
}
          

{
    'name': "Willi",
    'friend': {
        'name' : 'Robert'
    }
}
absolute refs: 

{ '$ref': 'http://data.com/person/#'}

referencing external json documents


{
    'type' : 'object',
    'properties': {
        'name' : {'type':'string'}
        'friendId' : {'type' : 'number'}
    },
    'links': {
        'rel':'friend',
        'href':'/people/{friendId}'
    }
}
          

{
    'name': 'Willi',
    'friendId': 5
}

type discriminator in gform schema

{
'type' : 'multi-object',
'typeProperty': '__type',
'groups': [
    {
        'code':'Car',
        'attributes':[
            {'code':'name', 'type':'string'}]
    },{
        'code':'Truck',
        'attributes':[
            {'code':'size', 'type':'int'}]
    }
]}


{
'__type': 'Car',
'name': 'Dudu'
}
